# Satisfaction, SEO & Growth (Skills 4, 5, 6)

Viewer satisfaction inference, SEO optimization, and growth pattern detection.

## Instructions

### Skill 4: Viewer Satisfaction Signals

1. **Satisfaction Index**:
   ```
   SI = (Engagement_Quality x 0.6) + (Retention_Proxy x 0.3) + (Implementation_Success x 0.1)
   ```
   - 80+: Very satisfied, 60-80: Satisfied, <40: Dissatisfied

2. **Content Clarity Score**: `Question_Density - Confusion_Ratio + Implementation_Stories`
   - High = Clear, actionable content

3. **Return Viewer Indicator**: `(Commenters in 2+ videos / Total Unique) x 100`

### Skill 5: Metadata & SEO

1. **Title Effectiveness Score**:
   ```
   TES = (Keyword_Placement x 0.4) + (Length_Optimization x 0.3) + (Hook_Strength x 0.3)
   ```
   - Optimal length: 50-60 chars
   - Keyword in first 30 chars: +10 points
   - Number hook ("7 Ways"): +10, Question: +8

2. **Description Quality Score**: First 150 chars must state benefit + primary keyword

3. **SEO Strength Index**: `(Title x 0.35) + (Description x 0.35) + (Tags x 0.3)`

### Skill 6: Growth Patterns

1. **Series Effectiveness**: `(Videos_in_Series x Avg_Engagement) / Total_Videos`
   - Series increase watch time by 89%

2. **Upload Consistency Impact**: Weekly = 156% higher growth

3. **Growth Drivers** (research-backed):
   - Upload Consistency: +156%
   - Community Engagement: +134%
   - Content Series: +89%
   - Multi-Format Strategy: +156%
