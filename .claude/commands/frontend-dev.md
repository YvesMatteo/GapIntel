# Frontend Development Server

Start the Next.js development server for local frontend development.

## Steps

1. **Navigate to frontend directory and install dependencies if needed**
```bash
cd /Users/yvesromano/AiRAG/gap-intel-website && npm install
```

2. **Start the development server**
```bash
cd /Users/yvesromano/AiRAG/gap-intel-website && npm run dev
```

3. **Access the app**
Open http://localhost:3000 in your browser.

## Notes
- The frontend uses Next.js 14 with App Router
- Hot reload is enabled by default
- Check terminal for TypeScript errors
- Press Ctrl+C to stop the server
