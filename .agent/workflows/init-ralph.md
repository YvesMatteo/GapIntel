# Initialize Ralph Loop

This workflow sets up the `PROMPT.md` and `IMPLEMENTATION_PLAN.md` files in the current directory to start a Ralph Loop session.

1. Create `PROMPT.md` from template.
2. Create `IMPLEMENTATION_PLAN.md` from template.

**Instructions**:
Run the following commands to copy the templates to the current directory:

```bash
cp .agent/skills/ralph-loop/templates/PROMPT.md PROMPT.md
cp .agent/skills/ralph-loop/templates/IMPLEMENTATION_PLAN.md IMPLEMENTATION_PLAN.md
```

Then, open `PROMPT.md` and fill in your objective.
